# Подключение к PostgreSQL базе данных KinoSwipe

## Данные для подключения:

```
Host: localhost (или IP вашего сервера)
Port: 5432
User: kinoswipe
Password: kinoswipe123
Database: kinoswipe
```

## Способ 1: Через командную строку (psql)

```bash
# Если PostgreSQL установлен локально
psql -h localhost -p 5432 -U kinoswipe -d kinoswipe

# Или через Docker
docker exec -it kinoswipe_db psql -U kinoswipe -d kinoswipe
```

## Способ 2: Через Docker контейнер

```bash
# Войти в контейнер PostgreSQL
docker exec -it kinoswipe_db bash

# Внутри контейнера подключиться к БД
psql -U kinoswipe -d kinoswipe
```

## Способ 3: Через графический клиент (pgAdmin, DBeaver, TablePlus)

1. Откройте ваш клиент PostgreSQL
2. Создайте новое подключение:
   - Host: localhost
   - Port: 5432
   - Database: kinoswipe
   - Username: kinoswipe
   - Password: kinoswipe123

## Полезные SQL команды:

```sql
-- Посмотреть все таблицы
\dt

-- Посмотреть все фильмы
SELECT * FROM movies;

-- Посчитать количество фильмов
SELECT COUNT(*) FROM movies;

-- Посмотреть все комнаты
SELECT * FROM rooms;

-- Посмотреть всех пользователей
SELECT * FROM users;

-- Посмотреть все свайпы
SELECT * FROM swipes;

-- Посмотреть все матчи
SELECT * FROM matches;
```

## Проверка, что БД работает:

```bash
# Проверить статус контейнера
docker ps | grep kinoswipe_db

# Проверить логи PostgreSQL
docker logs kinoswipe_db

# Проверить подключение
docker exec -it kinoswipe_db pg_isready -U kinoswipe
```
