# Проверка базы данных и загрузка фильмов

## Подключение к БД:

```bash
# Через Docker
docker exec -it kinoswipe_db psql -U kinoswipe -d kinoswipe
```

## Проверка наличия фильмов:

```sql
-- Посмотреть все фильмы
SELECT id, title, year, created_at FROM movies ORDER BY created_at DESC;

-- Посчитать количество фильмов
SELECT COUNT(*) FROM movies;

-- Если фильмов нет, можно загрузить тестовые данные
```

## Загрузка фильмов через скрипт:

```bash
# Если есть скрипт load_movies.go
cd /path/to/kinoswipe
go run scripts/load_movies.go
```

## Добавление фильмов через админ-панель:

1. Войдите как администратор (email: admin@example.com, password: admin123)
2. Перейдите в раздел "Фильмы"
3. Заполните форму и добавьте фильм

## Проверка через API:

```bash
# Проверить, что API возвращает фильмы
curl http://localhost:8080/api/v1/movies

# Или через браузер откройте:
http://localhost:8080/api/v1/movies
```

## Если фильмов нет в БД:

1. Проверьте, что миграции выполнены:
```bash
docker exec -it kinoswipe_db psql -U kinoswipe -d kinoswipe -c "\dt"
```

2. Добавьте фильмы через админ-панель или скрипт

3. Проверьте логи приложения:
```bash
docker logs kinoswipe_app
```
