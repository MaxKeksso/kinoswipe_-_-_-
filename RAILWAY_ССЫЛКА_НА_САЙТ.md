# Как получить ссылку на сайт и открыть его в интернете (Railway)

Подробная инструкция: как создаётся ссылка на ваш проект KinoSwipe на Railway и как посмотреть работающий сайт после того, как деплой завершён.

---

## 1. Когда проект уже задеплоен (логи прошли, всё создано и подключено)

После того как вы:
- создали проект на Railway;
- подключили GitHub-репозиторий;
- добавили PostgreSQL;
- настроили `DATABASE_URL` (Reference на PostgreSQL);
- прошли все логи сборки без ошибок;

сайт уже развёрнут и доступен в интернете. Остаётся получить ссылку и открыть её в браузере.

---

## 2. Где находится ссылка на сайт

### Шаг 1: Откройте проект на Railway

1. Зайдите на [railway.app](https://railway.app).
2. Войдите через GitHub (если ещё не вошли).
3. Откройте нужный проект (например, проект с KinoSwipe).

### Шаг 2: Выберите сервис приложения

В проекте обычно есть два сервиса:
- **kinoswipe** (или kinoswipe_-_-_-) — приложение (бэкенд + фронтенд);
- **PostgreSQL** — база данных.

Нужен сервис приложения, не база.

### Шаг 3: Откройте вкладку Settings

1. Кликните по карточке сервиса **приложения** (kinoswipe).
2. В левом меню выберите **Settings** (Настройки).

### Шаг 4: Секция Networking → Generate Domain

1. Прокрутите до секции **Networking** (или **Networking & Domains**).
2. Найдите блок **Public Networking** / **Generate Domain**.
3. Нажмите кнопку **Generate Domain** (или **+ New Domain**).

Если домен уже сгенерирован — вместо кнопки будет показана уже выданная ссылка.

### Шаг 5: Скопируйте ссылку

Railway выдаст домен вида:

```
https://kinoswipe-production-xxxx.up.railway.app
```

или:

```
https://ваш-проект-production-xxxx.up.railway.app
```

- Скопируйте эту ссылку.
- Это и есть адрес вашего сайта в интернете.

---

## 3. Как посмотреть сайт в браузере

1. Откройте браузер (Chrome, Safari, Firefox и т.п.).
2. Вставьте скопированную ссылку в адресную строку.
3. Нажмите Enter.

Сайт должен открыться. Пример:  
`https://kinoswipe-production.up.railway.app`

---

## 4. Если кнопки "Generate Domain" нет или домен не создаётся

Убедитесь, что:
- Вы находитесь в сервисе **приложения** (kinoswipe), а не в PostgreSQL.
- Вкладка **Settings** открыта.
- Секция **Networking** видна (возможно, нужно прокрутить вниз).

Если интерфейс изменился:
- Ищите раздел **Networking**, **Domains** или **Public URL**.
- Должна быть возможность создать или посмотреть публичный домен.

---

## 5. Где ещё можно увидеть ссылку

### Вкладка Deployments

1. В сервисе приложения выберите **Deployments**.
2. Откройте последний успешный деплой.
3. Справа или внизу иногда показывается **URL** / **Public URL** — это та же ссылка.

### Главная страница проекта

- В карточке сервиса приложения иногда отображается иконка ссылки или URL.
- Клик по ней открывает сайт.

---

## 6. Полезно знать

| Вопрос | Ответ |
|--------|-------|
| Ссылка меняется? | Нет, пока вы не удалите домен и не создадите новый. Обычно она постоянная. |
| HTTPS? | Да, Railway выдаёт домен с HTTPS. |
| Можно ли использовать свой домен? | Да, в **Settings → Networking** можно добавить свой домен. |
| После `git push` нужно ли что-то делать? | Нет. Railway сам пересобирает и деплоит приложение. Через 1–3 минуты изменения будут на сайте. |

---

## 7. Краткий чеклист

1. [railway.app](https://railway.app) → войти через GitHub.
2. Открыть проект → выбрать сервис **kinoswipe** (приложение).
3. **Settings** → **Networking** → **Generate Domain** (если домена ещё нет).
4. Скопировать ссылку вида `https://...up.railway.app`.
5. Вставить в браузер и открыть — это и есть ваш сайт в интернете.
