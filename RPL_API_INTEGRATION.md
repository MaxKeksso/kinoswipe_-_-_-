# Интеграция РПЛ данных

## Реализовано

РПЛ подгружается из **API-Football** (api-sports.io): лига 235, текущий сезон. Если переменная `API_FOOTBALL_KEY` не задана, используются статические данные (как раньше).

## Варианты получения реальных данных РПЛ

### 1. API-Football.com (Рекомендуется)

**Преимущества:**
- Предоставляет данные РПЛ
- Бесплатный план: 100 запросов в день
- Платный план: больше запросов и реал-тайм данные

**Регистрация:**
1. Зайдите на https://www.api-football.com/
2. Зарегистрируйтесь
3. Получите бесплатный API ключ (RapidAPI Key)
4. Добавьте переменную `API_FOOTBALL_KEY` на Railway

**Интеграция:**
```go
// В service/football_service.go добавить метод:
func (s *FootballService) fetchRPLFromAPIFootball() ([]FootballMatch, error) {
    // Использовать RapidAPI endpoint для РПЛ
    // https://api-football-v1.p.rapidapi.com/v3/fixtures?league=235&season=2024
}
```

### 2. Парсинг официального сайта РПЛ

**Преимущества:**
- Бесплатно
- Всегда актуальные данные

**Недостатки:**
- Может нарушать ToS сайта
- Требует регулярного обновления при изменении структуры сайта
- Может быть заблокирован

**Реализация:**
Использовать библиотеку для парсинга HTML (например, `goquery`)

### 3. Другие источники

- **SportsDataIO** - платный, но предоставляет РПЛ
- **TheSportsDB** - бесплатный, но ограниченные данные
- **SofaScore API** - неофициальный, может быть нестабильным

## Рекомендация

Для начала использовать **API-Football.com** через RapidAPI:
1. Бесплатный план достаточен для демо
2. Простая интеграция
3. Надежный источник данных
4. Поддерживает РПЛ

## Шаги для интеграции

1. Получить API ключ от API-Football.com
2. Добавить переменную `API_FOOTBALL_KEY` на Railway
3. Обновить `service/football_service.go` для использования нового API
4. Протестировать получение данных РПЛ

## Текущая реализация
